# SolanaScope

Real-time Solana intelligence API for AI agents.

## What is this?

SolanaScope provides machine-readable intelligence about the Solana ecosystem:

- **Protocol Health** — Real-time status of Jupiter, Kamino, Drift, Raydium
- **Yield Intelligence** — Best opportunities ranked by risk-adjusted returns  
- **Whale Alerts** — Large wallet movements that affect markets
- **Risk Scoring** — Protocol and position risk analysis

## Why?

Agents don't need dashboards. We need APIs.

Every agent building on Solana needs protocol data, but existing tools are built for humans (charts, buttons, pretty UIs). SolanaScope is pure infrastructure — JSON in, JSON out.
